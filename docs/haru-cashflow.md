# 하루 캐시플로우 및 요금 구조

> 최종 업데이트: 2026-04-30  
> 상태: 설계 확정 / UI 구현 진행 중

---

## 1. 목적

하루의 캐시플로우는 하루웨이 콘텐츠 탐색에서 시작해,  
상세 실행 정보 열람 → 맞춤 요청 → 하루이 상담으로 이어진다.

목표는 콘텐츠를 과도하게 잠그는 것이 아니라,  
**탐색과 신뢰 형성은 열어두고, 실행과 개인화에서 결제를 만드는 것**이다.

---

## 2. 기본 원칙

| 단계 | 상태 | 설명 |
|------|------|------|
| 탐색 | **무료** | 하루웨이 목록 · 흐름 · 요약 · 프로필 |
| 실행 | **유료** | 스팟 상세 가이드 · 저장 · 오프라인 보기 |
| 개인화 · 상담 | **고가** | 맞춤 요청 · 하루이 상담 |

---

## 3. 무료 제공 범위

- 하루웨이 목록 탐색
- 하루웨이 요약 (제목, 태그, 메타 정보)
- 하루 흐름 카드 (스팟명 + 이동 시간 + 순서)
- 한눈에 보는 하루 (RouteSummaryCard)
- 대표 이미지
- 스팟명 및 이동 순서 확인
- 하루이 프로필 보기
- 비슷한 하루웨이 보기

---

## 4. 유료 전환 지점

| 기능 | 연결 요금제 |
|------|------------|
| 스팟별 상세 실행 가이드 | 하루 바로 시작 |
| 사진 포인트 | 하루 바로 시작 |
| 주의사항 | 하루 바로 시작 |
| 다음 스팟 이동 세부 정보 | 하루 바로 시작 |
| 저장 / 오프라인 보기 | 하루 바로 시작 |
| 지도앱 열기 | 하루 바로 시작 |
| 내 일정에 맞게 요청 | 하루 맡기기 |
| 하루이 상담 | 하루 풀서비스 |

---

## 5. 요금제별 제공 범위

### 하루 바로 시작 — ₩29,000

**역할**: 완성된 하루웨이를 그대로 따라갑니다.

**포함**:
- 전체 스팟별 실행 가이드
- 사진 포인트
- 주의사항
- 다음 스팟 이동 정보 (시간 · 거리 · 이동 방식)
- 저장 및 오프라인 보기
- 지도앱 열기 링크

**제외**:
- 하루이 개인 상담
- 맞춤 루트 수정
- 채팅

**CTA**: "하루 바로 시작"  
**사용 지점**: 하루웨이 상세 화면 내 잠금 카드

---

### 하루 맡기기 — ₩59,000

**역할**: 내 날짜·취향에 맞게 하루이가 루트를 조정합니다.

**포함**:
- 여행 날짜 반영
- 시작 위치 / 종료 위치 반영
- 관심사 / 피하고 싶은 것 반영
- 동행 인원 반영
- 언어 / 이동 선호 반영
- 하루이의 1회 답변 또는 1회 수정
- 요청 상태 추적

**제외**:
- 실시간 채팅
- 무제한 상담

**CTA**: "내 일정에 맞게 요청"  
**사용 지점**: 하루웨이 상세 잠금 카드, 하루이 프로필, 요금 페이지

---

### 하루 풀서비스 — ₩119,000

**역할**: 하루이와 상담하며 하루를 세밀하게 조율합니다.

**포함**:
- 하루이 상담 (요청 기반, 24시간 내 답변)
- 맞춤 하루 설계
- 복수 후보 루트 제안
- 여행 전 최종 확인
- 일정 중간 조정 요청

**제외 (명시 필수)**:
- 긴급 상황 대응
- 의료 · 법률 · 통역 서비스
- 24시간 실시간 보호

**CTA**: "하루이와 조율하기"  
**사용 지점**: 하루이 프로필, 요청 플로우 내 업셀, 요금 페이지

> ⚠️ 하루 풀서비스는 실시간 무제한 채팅이 아닙니다.  
> "하루이 상담", "24시간 내 답변", "요청 기반 조율"로 표현합니다.

## 6. 하루웨이 상세 화면 결제 구조

### 무료 공개 영역

1. Hero
2. 하루 흐름 카드 (스팟명 + 이동 시간 + 순서)
3. 한눈에 보는 하루 (RouteSummaryCard)
4. 먼저 알고 가면 좋은 점 (route_highlights)
5. 하루이 요약 카드 (우측 Sticky)
6. 비슷한 하루웨이

### 유료 잠금 영역

스팟 목록 프리뷰 (번호 · 이름만) 노출 후, 아래 잠금 카드로 전환:

```
스팟별 상세 가이드는
하루 바로 시작 후 확인할 수 있습니다.

포함:
✓ 스팟별 실행 가이드
✓ 사진 포인트
✓ 주의사항
✓ 다음 스팟 이동 정보
✓ 저장 및 오프라인 보기

[하루 바로 시작  ₩29,000]
[내 일정에 맞게 요청]
```

## 7. 요청 플로우

```
Step 1. 요청 조건 입력
  ├ 여행 날짜
  ├ 시작 위치 / 종료 위치
  ├ 동행 인원
  ├ 관심사 / 피하고 싶은 것
  ├ 이동 선호 / 사용 언어
  └ 요청 메모

Step 2. 상품 선택
  ├ 하루 맡기기 ₩59,000 — 하루이가 조건에 맞게 루트 조정
  └ 하루 풀서비스 ₩119,000 — 하루이와 상담하며 세밀하게 조율

Step 3. 결제

Step 4. 요청 상태
  요청 접수 → 하루이 확인 중 → 하루웨이 조정 중 → 답변 도착 → 확정 완료
```

> 결제 전 요청 조건을 먼저 입력하게 한다.  
> 조건 입력 과정이 구매 의도를 높인다.

## 8. 하루이 정산 구조

```
여행자 결제
  ↓
플랫폼 결제 보관
  ↓
하루웨이 제공 또는 요청 완료
  ↓
여행자 확인
  ↓
하루이 정산
  ↓
플랫폼 수수료 확보
```

### 상품별 정산 기준

| 상품 | 정산 시점 |
|------|----------|
| 하루 바로 시작 | 콘텐츠 열람 완료 시 |
| 하루 맡기기 | 요청 완료 후 |
| 하루 풀서비스 | 상담/설계 제공 완료 후 |

### 정산율 (초안)

- 하루이: 80%
- 플랫폼: 20%
- 초기 하루이 혜택: 첫 3개월 수수료 우대 가능 (정책 확정 시 공지)

> ⚠️ 수수료 정책이 확정되지 않은 경우 UI에 "수수료 우대"로만 표현한다.  
> "수수료 0%"와 "80% 정산"을 동시에 노출하지 않는다.

## 9. 운영 리스크와 제한 문구

### 금지 표현

| 금지 | 대체 |
|------|------|
| 실시간 무제한 채팅 | 요청 기반 하루이 상담 |
| 24시간 보호 | 제공 불가 — 명시 제외 |
| 긴급 대응 | 제공 불가 — 명시 제외 |
| 무조건 만족 보장 | 환불 기준 별도 공지 |
| 확정되지 않은 수익/정산 보장 | 정책 확정 후 공지 |

### 의무 고지 문구

하루 풀서비스는 긴급 대응, 의료·법률·통역 서비스를 대체하지 않습니다.
하루이 상담은 24시간 실시간 보호 서비스가 아닙니다.
